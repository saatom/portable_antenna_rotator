## Overview
This project was made for the Arctic Amateur Radio Club (KL7KC) in Fairbanks, Alaska to enable easy radio communication with satellites and ground-based targets alike with a single, GPS-enabled device with 400+ degrees of azimuth and 180 degrees of elevation control.

The code is intended to be run on a Raspberry Pi 4b with the relevant Python packages installed.
